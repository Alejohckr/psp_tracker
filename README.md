# psp_tracker
Personal Software Process tracker for university courses — built with React, Vite, Tailwind CSS and Supabase.

Web app for tracking the Personal Software Process (PSP) across a 9-week university course.
  Covers weekly planning, time logging with automatic deviation (desfase), exercise tracking with
  LOC metrics, defect logging (PSP standard types 10–100), test case management, and visual
  analytics — all scoped per user via Supabase Row Level Security.

  Stack: React 19 · Vite · Tailwind CSS v4 · Supabase (Auth + PostgreSQL)
